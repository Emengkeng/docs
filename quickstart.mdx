## For Users: Subscribe in 3 Steps

Get started with your first on-chain subscription and start earning yield on idle funds.

<Steps>
  <Step title="Create Your Subscription Wallet">
    ```javascript
    import { SubscriptionProtocol } from '@subscription-protocol/sdk';
    
    const protocol = new SubscriptionProtocol();
    
    // Connect your Solana wallet
    await protocol.connectWallet();
    
    // Create subscription wallet (PDA)
    const wallet = await protocol.createSubscriptionWallet({
      mint: 'USDC_MINT_ADDRESS'
    });
    
    console.log('Wallet created:', wallet.publicKey);
    ```
    
    <Note>
      Your subscription wallet is a Program Derived Address (PDA) that you control. No one else can access your funds.
    </Note>
  </Step>
  
  <Step title="Deposit Funds & Enable Yield (Optional)">
    ```javascript
    // Deposit USDC to your subscription wallet
    await protocol.deposit({
      amount: 1000, // 1000 USDC
      wallet: wallet.publicKey
    });
    
    // Enable yield earning (optional but recommended)
    await protocol.enableYield({
      wallet: wallet.publicKey,
      strategy: 'MarginfiLend' // Options: MarginfiLend, KaminoLend, SolendPool, DriftDeposit
    });
    
    console.log('Funds deposited and yield enabled!');
    ```
    
    <Tip>
      With yield enabled, your idle funds earn 5-8% APY through integrated DeFi protocols while waiting to pay subscriptions.
    </Tip>
  </Step>
  
  <Step title="Subscribe to a Service">
    ```javascript
    // Browse available plans
    const plans = await protocol.getMerchantPlans('MERCHANT_ADDRESS');
    
    // Subscribe to a plan
    const subscription = await protocol.subscribe({
      planId: 'premium-monthly',
      wallet: wallet.publicKey
    });
    
    console.log('Subscription active:', subscription.publicKey);
    ```
    
    <Check>
      Done! Payments will execute automatically on schedule. You can cancel anytime from your dashboard.
    </Check>
  </Step>
</Steps>

## For Merchants: Accept Payments in 3 Steps

Start accepting recurring crypto payments without blockchain complexity.

<Steps>
  <Step title="Get Your API Keys">
    ```bash
    # Sign up and get your API credentials
    curl -X POST https://api.subscription-protocol.com/v1/auth/register \
      -H "Content-Type: application/json" \
      -d '{
        "email": "merchant@example.com",
        "businessName": "Your Business"
      }'
    ```
    
    You'll receive:
    - API Key
    - Webhook Secret
    - Merchant Wallet Address
    
    <Warning>
      Keep your API key secure. Never expose it in client-side code.
    </Warning>
  </Step>
  
  <Step title="Create a Subscription Plan">
    ```javascript
    const axios = require('axios');
    
    const response = await axios.post(
      'https://api.subscription-protocol.com/v1/plans',
      {
        planId: 'premium-monthly',
        planName: 'Premium Plan',
        feeAmount: 9.99, // in USDC
        paymentInterval: 2592000, // 30 days in seconds
        mint: 'USDC_MINT_ADDRESS'
      },
      {
        headers: {
          'Authorization': `Bearer ${YOUR_API_KEY}`,
          'Content-Type': 'application/json'
        }
      }
    );
    
    console.log('Plan created:', response.data.planAddress);
    ```
    
    <Info>
      You can create multiple plans with different pricing tiers and intervals (daily, weekly, monthly, yearly).
    </Info>
  </Step>
  
  <Step title="Integrate into Your App">
    ```javascript
    // Install the SDK
    npm install @subscription-protocol/sdk
    
    // In your checkout page
    import { SubscriptionProtocol } from '@subscription-protocol/sdk';
    
    const protocol = new SubscriptionProtocol({
      apiKey: process.env.API_KEY,
      environment: 'mainnet' // or 'devnet' for testing
    });
    
    // When user clicks subscribe
    async function handleSubscribe() {
      try {
        const subscription = await protocol.createSubscription({
          planId: 'premium-monthly',
          userWallet: userWalletAddress
        });
        
        // Redirect to success page
        window.location.href = `/success?subscription=${subscription.id}`;
      } catch (error) {
        console.error('Subscription failed:', error);
      }
    }
    ```
    
    <Check>
      That's it! Payments will flow directly to your merchant wallet automatically.
    </Check>
  </Step>
</Steps>

## For Developers: Build on the Protocol

Integrate directly with our Solana smart contracts for maximum flexibility.

<Steps>
  <Step title="Set Up Your Environment">
    ```bash
    # Install Solana CLI
    sh -c "$(curl -sSfL https://release.solana.com/stable/install)"
    
    # Install Anchor
    cargo install --git https://github.com/coral-xyz/anchor avm --locked --force
    avm install latest
    avm use latest
    
    # Clone our examples repo
    git clone https://github.com/your-org/subscription-protocol-examples
    cd subscription-protocol-examples
    ```
  </Step>
  
  <Step title="Understand the Program Structure">
    ```rust
    // Our main program instructions:
    
    // User instructions
    create_subscription_wallet()  // Create PDA wallet
    deposit_to_wallet()          // Add funds
    enable_yield()               // Enable yield earning
    subscribe_with_wallet()      // Subscribe to plan
    withdraw_from_wallet()       // Withdraw idle funds
    cancel_subscription_wallet() // Cancel subscription
    claim_yield_rewards()        // Claim yield earnings
    
    // Merchant instructions
    register_merchant()          // Create subscription plan
    execute_payment_from_wallet() // Process payment (anyone can call)
    ```
    
    <Note>
      All program accounts and instructions are documented in our [Smart Contract Reference](/developers/contracts).
    </Note>
  </Step>
  
  <Step title="Make Your First Call">
    ```typescript
    import * as anchor from "@coral-xyz/anchor";
    import { Program } from "@coral-xyz/anchor";
    import { SubscriptionProtocol } from "../target/types/subscription_protocol";
    
    const provider = anchor.AnchorProvider.env();
    anchor.setProvider(provider);
    
    const program = anchor.workspace.SubscriptionProtocol as Program<SubscriptionProtocol>;
    
    // Create subscription wallet
    const [subscriptionWallet] = await anchor.web3.PublicKey.findProgramAddress(
      [
        Buffer.from("subscription_wallet"),
        provider.wallet.publicKey.toBuffer(),
        mint.toBuffer()
      ],
      program.programId
    );
    
    await program.methods
      .createSubscriptionWallet()
      .accounts({
        subscriptionWallet,
        user: provider.wallet.publicKey,
        mint: mint,
        // ... other accounts
      })
      .rpc();
    ```
  </Step>
</Steps>
